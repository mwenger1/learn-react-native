I Really Wanted to learn RN

The platform promises a lot:

  • Build mobile applications that feel more "native" than other platforms (Phonegap, Titaninum)
  • 85% code reuse while shipping to Google Play and iOS app stores
  • It's Javascript. Don't need to learn a language

Time to Dive In and Learn

A friend was working on a React Native application and I offered to help in my spare time.

          
            git clone [FRIENDS-REPO]
            npm install -g react-native-cli
            npm install
            react-native run-ios

BOOM! I'm an iOS developer!
